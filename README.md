- 📈 [V1 Interactive — Process Capability](https://automotive-process-capability.streamlit.app/) — CSV upload, pasted measurements, Cp/Cpk scenario analysis, visualizations, out-of-specification review, and downloadable Word reports.
- 🧠 [V2 — Manufacturing Intelligence](https://manufacturing-process-intelligence.streamlit.app/) — session production records, FPY/RTY/OEE, BM25 engineering-lessons retrieval, evidence-grounded investigation briefs, and integrated reporting.
